A begginer programming fundamentals project including game development introduction -XONIX
A complete functional implementation of the classic Xonix Game developed using the SFML (Simple and Fast Multimedia Library) in C++. This project was developed as a team assignment with multiple gameplay features, difficulty levels, scoring systems, powerups, and both single-player and multiplayer support.

Tech Stack
Language: C++
Library: SFML
Concepts Used:
File Handling
Arrays & Grids
Collision Detection
Game State Management
Event Handling


When the game starts, players are welcomed with:
“Welcome to Xonix!”
Players can choose between:
1 → Single Player Mode
2 → Two Player Mode
